## 🖥️ What portmap-HDL does

portmap-HDL is a command-line tool for Verilog work. It helps you:

- extract port lists from Verilog files
- show port definitions in a clear format
- build a file tree for a hardware repo
- make large HDL projects easier to browse
- keep project files easier to find

It is made for people who work with hardware projects and want a faster way to inspect Verilog code and repo layout.
